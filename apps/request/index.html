Request Data

Individual level sequence data and/or variant calls can be requested.

The Request Process

  1. To apply for access, fill in the GoNL Data Access Request Application v1.1 (PDF, 132KB) and send to gonl@bbmri.nl
  2. Your research request will be evaluated by the GoNL steering committee.
  3. If your request has been successful, you will be sent the Data Access Agreement (PDF, 507KB). This agreement requires signatures from the applicant and a legal representative of the applicantâ€™s institution.
  4. You will be granted access to the data using a suitable method. At the time of publication, add the suitable acknowledgement. More information can be found in the GoNL Publication Acknowledgment document (PDF, 139KB).
